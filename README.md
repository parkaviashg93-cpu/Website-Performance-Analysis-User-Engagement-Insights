# 📊 Website Performance Analysis and User Engagement Insights

## 📌 Project Overview

This project focuses on analyzing website performance data to understand user behavior, engagement patterns, and performance issues. The goal is to derive actionable insights that help improve website speed, user experience, and overall engagement.

---

## 🎯 Objective

* Analyze website performance across devices, browsers, and locations
* Identify factors affecting page load time
* Understand user engagement patterns
* Clean and preprocess real-world data
* Create meaningful visualizations
* Provide data-driven recommendations

---

## 🧠 Business Goals

* Improve user engagement
* Reduce bounce rate
* Increase conversion rate
* Optimize website performance

---

## 📂 Dataset Information

* **Source:** https://amanxai.com
* **Year:** 2024
* **Domain:** E-commerce (Web Analytics)

### 📊 Features Included:

* Users
* Sessions
* Engaged Sessions
* Engagement Rate
* Events per Session
* Event Count
* Average Engagement Time
* Marketing Channel
* Date & Hour

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas

## 🔍 Data Preprocessing

* Handled missing values
* Removed duplicates
* Fixed inconsistent data formats
* Prepared dataset for analysis

---

## 📈 Exploratory Data Analysis (EDA)

* Distribution of page load times
* Device-wise performance comparison
* Browser performance analysis
* Engagement trends across channels
* Time-based traffic patterns

---

## 📊 Visualizations

* Histograms
* Boxplots
* Bar charts
* Time series plots

---

## 💡 Key Insights

* 📱 Mobile devices have higher page load times than desktops
* 🌐 Browser performance varies (Chrome performs better)
* ⚠️ Some pages show unusually high load times (outliers)
* 📉 High sessions don’t always mean high engagement
* ⏰ User activity fluctuates based on time

---

## 🚀 Recommendations

* Optimize website for mobile devices
* Improve performance of slow-loading pages
* Ensure cross-browser compatibility
* Monitor and fix performance outliers
* Enhance user experience to increase engagement
